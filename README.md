## DIU Event Junction: An event management system (role-based)
DIU Event Junction is a full-stack project. It is one of my best university projects (group project) and  I led the team to complete it.

### Tech used: 
**Frontend:** React(Vite), Tailwind CSS, Javascript, HTML, Redux-toolkit   
**Backend:** Node(Express.js), MongoDB    
**UI Design:** Figma
